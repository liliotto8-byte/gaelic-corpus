# Корпус и обработка шотландского гэльского


## Что где лежит

**preprocess.py** — пайплайн обработки: алгоритм очистки и лемматизатор. Читает сырые тексты и сохраняет очищенные и лемматизированные версии (по одному txt-файлу на текст).

**corpus_raw.zip** — необработанный корпус: исходные тексты до очистки. Каждый текст лежит отдельным txt-файлом с исходным названием (wiki_*, bible_*, poetry_*, dict_*)

**corpus_clean.zip** — очищенный корпус: нормализация, очистка и фильтр языка, без лемматизации. По одному txt-файлу на текст

**corpus_lemmatized.zip** — лемматизированный корпус: очистка, удаление стоп-слов и лемматизация. По одному txt-файлу на текст. Именно эти тексты используются для построения эмбеддингов

**lexicon/clann.txt** — словарь словоформ гэльского (hunspell-gd). Используется лемматизатором и фильтром языка

**embeddings/gaelic_svd_dictionary.npy** — SVD-эмбеддинги слов

**embeddings/gaelic_fasttext_dictionary.npy** — fastText-эмбеддинги слов
